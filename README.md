## 🎯 ¿Qué es este repositorio?

Este repositorio es la **bitácora pública de un bootcamp autodidacta de 16 semanas** diseñado para transformar conocimientos previos en un perfil competitivo de **Data Analyst Jr. especializado en Retail Analytics**.

No es un curso comprado. No es contenido genérico. Es un **programa de entrenamiento real**, construido con disciplina militar, enfocado en un solo objetivo:

> **Ser contratado como Data Analyst Jr. con un portfolio que supere al de bootcamps de $10,000+.**

```
NO es solo aprender → Es DEMOSTRAR que sabes → Para SER CONTRATADO.
```

```
Lunes–Viernes:   8:00 AM – 4:00 PM  (8 horas/día)
Sábado:          8:00 AM – 1:00 PM  (5 horas)

Total semanal:   ~45 horas
Total programa:  ~720 horas
```

Cada carpeta semanal contiene:

## 🚀 Cómo Usar Este Repositorio

### Si eres reclutador / hiring manager:
➡️ Ve directo a la carpeta de cada semana y revisa los entregables técnicos. Cada `README.md` tiene el resumen ejecutivo del trabajo realizado.

### Si estás aprendiendo Data Analytics:
➡️ Clona este repo y sigue la estructura semana a semana. Los archivos de teoría y práctica son recursos completos y autocontenidos.

### Si usas NotebookLM:
1. Carga los 5 archivos de cada carpeta semanal en un cuaderno
2. Usa el `README.md` como archivo ancla (cárgalo primero)
3. Haz preguntas como *"¿Cuáles son los ejercicios de nivel avanzado?"* o *"¿Cómo hago un Unpivot?"*
